The setup — step by step

01

Download the files

On the old phone, connect to WiFi and download the basecamp zip.

⬇ Download feelfamous-basecamp.zip

When it downloads, your phone will offer to unzip it. Do that. It'll create a folder called feelfamous-basecamp.

02

Move the files

Open your file manager. Find the feelfamous-basecamp folder (usually in Downloads). Move it to Internal Storage and rename it basecamp.

It needs to be at the root of Internal Storage — not buried inside another folder. Internal Storage → basecamp → index.html. That's the structure.

03

Install the server app

Go to Play Store. Search for HTTP File Server. Install the one by Slowscript. It's free.

Open the app. Tap the folder icon. Navigate to Internal Storage → basecamp. Select it. Hit Start.

The app will show you an IP address — something like 192.168.x.x:8080. Write it down or screenshot it. You'll need it in a moment.

04

Name your hotspot FeelFamous

Go to Settings → Network → Mobile Hotspot. Turn it on. Tap the name and change it to FeelFamous. No password — leave it open. This is intentional. You want people to connect.

Keep the HTTP File Server running in the background while the hotspot is on.

05

Make the QR code

On another device connected to your FeelFamous hotspot, open a browser and go to:

http://[your IP]:8080/files/qr.html

Paste in the full URL: http://[your IP]:8080/files/index.html

Generate the QR. Save it. That's what people scan to go straight to the basecamp.

The IP can change when you restart. Check the app each time and regenerate the QR if it's different.

Troubleshooting

App can't see my files — check Internal Storage permissions. Go to Settings → Apps → HTTP File Server → Permissions → Allow management of all files.

Browser shows a / with no content — go to the full URL: http://[IP]:8080/files/index.html

IP has changed since last time — check the HTTP File Server app for the current IP. Regenerate the QR.

People can't connect — make sure the hotspot is named FeelFamous and has no password.
